## О проекте

Современные магистерские программы предлагают высокую вариативность: студенты сами выбирают элективные курсы, а их уровень сложности адаптируется в зависимости от подготовки. Из-за этого возникает проблема: студент не всегда видит прямую связь между выбранным курсом и итоговыми компетенциями, требуемыми для его специальности. 

**Наша цель** — обеспечить прозрачность образовательного процесса и гарантировать, что к моменту выпуска студент получит необходимый объем навыков для успешной работы.

Этот сервис представляет собой интерактивную платформу для студентов магистратуры, которая помогает планировать обучение на два года вперед, визуализирует прогресс и подбирает идеальные курсы для закрытия "пробелов" в знаниях.

## Ключевой функционал

Наш прототип веб-приложения включает в себя следующие возможности:

* **Визуализация навыков:** Наглядное отображение текущего уровня компетенций на основе уже пройденных обязательных и элективных курсов.
* **Умные рекомендации:** Система автоматически анализирует дефицит навыков и предлагает курсы, которые наиболее эффективно их восполнят.
* **Помощник выбора курсов:** Удобный интерфейс для формирования расписания на текущий семестр с учетом матрицы компетенций.
* **Функция «Что, если?»:** Интерактивное моделирование изменения итогового профиля навыков при гипотетической смене специальности или выборе другого трека.

## 🛠 Технологический стек

* **Frontend:** C# Avalonia.UI
* **Backend:** C# ASP.NET
* **База данных:** PostgreSQL
